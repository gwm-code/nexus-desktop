What is Nexus Desktop?

  • Desktop app for the Nexus CLI (AI coding assistant)
  • Connects to remote servers via SSH or runs locally
  • Chat with AI, run commands, manage swarm tasks
  • Built with Tauri 2 + React + Zustand
  • Inherited 22,342 conversations from OpenClaw

How Connection Works

  • Step 1: Open Settings ‚Üí Connection tab
  • Step 2: Enter SSH details (host, port, auth)
  • Step 3: Click Connect
  • Step 4: Status bar shows "Connected (SSH)"
  • Step 5: All commands now execute on remote
  • Auto-reconnects if connection drops

Setting Up Providers

  • Google (OAuth): Uses gemini-cli auth automatically
  • OpenRouter: Enter API key (sk-or-v1-...)
  • OpenCode: Enter API key
  • Keys stored securely (OS keyring or plaintext fallback)
  • Models auto-fetched from provider API
  • Test connection before using

How It Works Under the Hood

  • Frontend: React + Zustand (TypeScript)
  • Backend: Tauri (Rust) + SSH
  • Bridge: execute_nexus_bridge calls CLI
  • CLI: Nexus binary on remote/local server
  • Providers: Google, OpenRouter, OpenCode
  • All commands use --json flag for clean output

üí° Tips & Tricks

  • API keys clear from UI after saving (security feature)
  • Models list works without valid API key (public endpoint)
  • Test connection fails? Check your API key format
  • SSH + Local: Desktop tries SSH first, falls back to local
  • Chat responses are markdown-formatted
  • Swarm mode great for multi-file refactors

üîß Troubleshooting

  • Raw JSON in chat? Update to latest AppImage
  • Models empty? Check CLI version on server
  • Connection fails? Verify SSH credentials
  • API key won't save? Provider auto-created on first save
  • ANSI codes? Update CLI to version with --json fix
